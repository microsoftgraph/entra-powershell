# Get-EntraDomain

## SYNOPSIS
Gets a domain.

## SYNTAX

### GetQuery (Default)
```
Get-EntraDomain [<CommonParameters>]
```

### GetById
```
Get-EntraDomain -Name <String> [<CommonParameters>]
```

## DESCRIPTION
The Get-EntraDomain cmdlet gets a domain in Microsoft Entra ID.
