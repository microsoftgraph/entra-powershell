# Get-EntraBetaDomain

## SYNOPSIS
Gets a domain.

## SYNTAX

### GetQuery (Default)
```
Get-EntraBetaDomain [<CommonParameters>]
```

### GetById
```
Get-EntraBetaDomain -Name <String> [<CommonParameters>]
```

## DESCRIPTION
The Get-EntraBetaDomain cmdlet gets a domain in Azure Active Directory (AD).
