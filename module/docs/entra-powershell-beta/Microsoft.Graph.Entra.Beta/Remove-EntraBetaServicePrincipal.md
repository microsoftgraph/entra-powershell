# Remove-EntraBetaServicePrincipal

## Synopsis

## Syntax

```
Remove-EntraBetaServicePrincipal -ObjectId <String> [<CommonParameters>]
```

## Description
the Remove-EntraBetaServicePrincipal cmdlet removes a service principal in Azure Active Directory (AD).
