# Get-EntraBetaMSApplication

## SYNOPSIS
Retrieves the list of applications within the organization.

## SYNTAX

### GetQuery (Default)
```powershell
Get-EntraBetaMSApplication 
    [-Filter <String>] 
    [-All <Boolean>] 
    [-Top <Int32>] 
 [<CommonParameters>]
```

### GetVague
```powershell
Get-EntraBetaMSApplication 
    [-SearchString <String>] 
    [-All <Boolean>] 
 [<CommonParameters>]
```

### GetById
```powershell
Get-EntraBetaMSApplication 
    -ObjectId <String> 
    [-All <Boolean>] 
 [<CommonParameters>]
```

## DESCRIPTION
Retrieves the list of applications within the organization.
With an ObjectId argument, it can retrieve the properties of the application object associated with the ObjectId.
