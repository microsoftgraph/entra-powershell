# Get-EntraDomainNameReference

## SYNOPSIS
This cmdlet retrieves the objects that are referenced with a given domain name.

## SYNTAX

```powershell
Get-EntraDomainNameReference 
    -Name <String> 
    [<CommonParameters>]
```

## DESCRIPTION
This cmdlet retrieves the objects that are referenced with a given domain name.
