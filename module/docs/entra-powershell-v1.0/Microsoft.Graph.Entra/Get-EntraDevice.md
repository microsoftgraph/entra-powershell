# Get-EntraDevice

## SYNOPSIS
Gets a device from Microsoft Entra ID.

## SYNTAX

### GetQuery (Default)
```powershell
Get-EntraDevice 
 [-Top <Int32>]
 [-All <Boolean>]
 [-Filter <String>]
 [<CommonParameters>]
```

### GetByValue
```powershell
Get-EntraDevice 
 [-SearchString <String>]
 [-All <Boolean>]
 [<CommonParameters>]
```

### GetById
```powershell
Get-EntraDevice 
 -ObjectId <String>
 [-All <Boolean>]
 [<CommonParameters>]
```

## DESCRIPTION
The Get-EntraDevice cmdlet gets a device from Microsoft Entra ID.
