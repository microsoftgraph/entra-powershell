# Enable-EntraAzureADAlias

## Synopsis

enables aliases for AzureAD modules.

## Syntax

```powershell
Enable-EntraAzureADAlias
```

## Description

enables aliases for Azure AD modules.

## Examples

### Example 1

```powershell
Enable-EntraAzureADAlias
```

Enables all Azure AD prefixes for the current PowerShell session.
