# Get-EntraUserAppRoleAssignment

## SYNOPSIS
Get a user application role assignment.

## SYNTAX

```powershell
Get-EntraUserAppRoleAssignment
 -ObjectId <String>
 [-All]
 [-Top <Int32>]
 [<CommonParameters>]
```

## DESCRIPTION
The Get-EntraUserAppRoleAssignment cmdlet gets a user application role assignment.

## EXAMPLES

PS C:\> $UserId = (Get-EntraUser -Top 1).ObjectId
