# Get-EntraDeletedApplication

## SYNOPSIS

Retrieves the list of previously deleted applications.

## SYNTAX

### GetQuery (Default)

```powershell
Get-EntraDeletedApplication 
 [-Filter <String>] 
 [-All] 
 [-Top <Int32>] 
 [<CommonParameters>]
```

### GetByValue

```powershell
Get-EntraDeletedApplication 
 [-SearchString <String>] 
 [-All] 
 [<CommonParameters>]
```

## DESCRIPTION

Retrieves the list of previously deleted applications.

## EXAMPLES

Connect-Entra -Scopes 'Application.Read.All'
