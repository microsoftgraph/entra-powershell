# Get-EntraDeletedApplication

## SYNOPSIS
Retrieves the list of previously deleted applications.

## SYNTAX

### GetQuery (Default)
```powershell
Get-EntraDeletedApplication 
    [-Filter <String>] 
    [-All <Boolean>] 
    [-Top <Int32>] 
    [<CommonParameters>]
```

### GetValue
```powershell
Get-EntraDeletedApplication 
    [-SearchString <String>] 
    [-All <Boolean>] 
    [<CommonParameters>]
```

## DESCRIPTION
Retrieves the list of previously deleted applications.
