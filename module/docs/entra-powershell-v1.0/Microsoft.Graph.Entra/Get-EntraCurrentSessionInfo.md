# Get-EntraCurrentSessionInfo

## SYNOPSIS
This cmdlet returns the current session state.

## SYNTAX

```powershell
Get-EntraCurrentSessionInfo 
    [<CommonParameters>]
```

## DESCRIPTION
This cmdlet returns the current session state.

## EXAMPLES
